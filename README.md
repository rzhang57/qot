# qot
frictionless real-time collaborative team workspace

## what
- real-time team collaboration platform backed by websockets and Microsoft's SignalR
- temporary and session-based; frictionless with no setup overhead
- import/export documents to share directly from qot

## setup

**server (.net)**
```bash
cd qot
dotnet restore
dotnet run
```

**client (react)**
```bash
cd qot-client
npm install
npm run dev
```
